#  Inkwell

Inkwell is a clean and elegant writing platform built with HTML, CSS, and JavaScript. It provides a distraction-free environment for reading, writing, and exploring beautiful content.

##  Features

-  Clean and modern design
-  Responsive layout
-  Fast and lightweight
-  Beautiful typography
-  Works on desktop and mobile

##  Built With

- HTML5
- CSS3
- JavaScript

##  Getting Started

1. Clone this repository:

```bash
git clone https://github.com/yourusername/inkwell.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

##  Project Structure

```
Inkwell/
│── index.html
│── style.css
```
